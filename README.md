# 迫击炮弹道计算器

一个基于高德地图 API 的迫击炮弹道计算工具，用于计算迫击炮射击参数并在地图上显示弹道轨迹。

## 功能特点

- 🗺️ **地图定位**：自动获取炮位坐标，支持手动输入坐标
- 🎯 **目标标记**：点击地图标记目标点
- 📐 **参数计算**：自动计算距离、方位角、仰角
- 🚀 **弹道轨迹**：显示黄色抛物线弹道轨迹，包含高度和速度信息
- ⏱️ **落地时间**：计算并显示炮弹落地时间
- 📱 **响应式设计**：支持手机、平板和 PC 设备
- 🛰️ **卫星地图**：支持切换卫星地图和普通地图
- 🔔 **气泡通知**：操作反馈通过悬浮气泡显示

## 支持的迫击炮型号

- 60mm 迫击炮
- 82mm 迫击炮
- 120mm 迫击炮

## 如何使用

1. 打开 `index.html` 文件
2. 点击「获取炮位坐标」按钮获取当前位置，或手动输入坐标
3. 在地图上点击标记目标点
4. 选择迫击炮型号和装药级别
5. 查看计算结果和弹道轨迹
