# 迫击炮弹道计算器

一个基于高德地图 API 的迫击炮弹道计算工具，用于计算迫击炮射击参数并在地图上显示射击信息。

## 功能特点

- 🗺️ **地图定位**：自动获取炮位坐标，支持手动输入坐标
- 🎯 **目标标记**：点击地图标记目标点
- 📐 **参数计算**：自动计算距离、方位角、仰角
- ⏱️ **落地时间**：计算并显示炮弹飞行时间
- 📊 **射程显示**：在地图上以炮位为圆心显示最大射程圆形
- 🔔 **超射程提示**：当目标超出射程时显示警告
- 📱 **响应式设计**：支持手机、平板和 PC 设备
- 🛰️ **卫星地图**：支持切换卫星地图和普通地图
- 🔔 **气泡通知**：操作反馈通过悬浮气泡显示

## 支持的迫击炮型号

- 60mm 迫击炮
- 82mm 迫击炮
- 120mm 迫击炮

每种型号支持 0-4 号装药，不同装药量影响射程和弹道。

## 如何使用

1. 打开 `index.html` 文件
2. 点击「获取炮位坐标」按钮获取当前位置
3. 在地图上点击标记目标点
4. 选择迫击炮型号和装药级别
5. 查看计算结果（距离、方位角、仰角、飞行时间）
